SPSA 2018 

This program:
1) generates DSA and RSA keys;
2) reads a file that contains a YAO's garbled circuit;
3) encrypts the file using RSA Public Key;
4) signs the file using DSA Private Key;
5) verifies the signature of the file using DSA Public Key;
6) decrypts the file using RSA Private Key;
7) evaluates the contained YAO's garbeld circuit;
